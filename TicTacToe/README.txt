/////////////////////////Installation/////////////////////////
Launch TicTacToe.sln located in ../TicTacToe/TicTacToe.sln
This program requires the 1.4.3 Bespoke.games.framework NuGet package
and it's dependencies.

Build the Solution/Project, it should be able to build any
of the configuration modes (DEBUG/Release @ WIN32/x64) using Windows 10.

////////////Controls/Instructions (Keyboard/Mouse)////////////
* ESC - quit
* Left Click - select position to place 'X' or 'O'
